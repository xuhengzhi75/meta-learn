# Changelog

## [1.0.0] - 2026-04-10

### Added
- Initial release
- 5 interactive learning modes: Deep Dive, Feynman Sparring, Socratic Drill, Instant Feedback, Cross-Domain Transfer
- 3-layer knowledge skeleton (core / important / advanced)
- Personalized learning path with daily tasks and weekly checkpoints
- Chinese and English bilingual support (auto-detects language)
- Feynman sparring: single-question Socratic approach (not exam-style barrage)
- Description optimized via automated trigger eval loop (train 100% / test 100% on 20 queries)
- Platform support: Claude Code, Friday Skills Hub, Cursor, Gemini CLI, agentskills.io
